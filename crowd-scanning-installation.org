* Context

This document provides a step-by-step installation guide for the *FACES 2 Crowd Scanning* application on NVIDIA Jetson Orin NX devices.

#+begin_quote
The recipe uses absolute paths and can be executed from any directory on the Jetson device.
All code blocks are designed to be idempotent where possible.

#+begin_src bash :results output replace :exports both :dir /sudo:: :session f2cs :var LICENSE_KEY=license-key
  set -euo pipefail

  sudo tee /etc/systemd/system/f2_crowd_scanner.service > /dev/null << SERVICEEOF
[Unit]
Description=F2 Crowd Scanning FaceMe Service
After=network.target
StartLimitBurst=3
StartLimitIntervalSec=60
OnFailure=f2_crowd_scanner-email@%n

[Service]
Type=simple
WorkingDirectory=$PROJECT_DIR
ExecStart=/usr/bin/script -q -c "$VENV_DIR/bin/python $PROJECT_DIR/main.py" /dev/null
Restart=always
RestartSec=10
Environment=FACEME_LICENSE_KEY=$LICENSE_KEY
StandardOutput=journal
StandardError=journal

[Install]
WantedBy=multi-user.target
SERVICEEOF

  echo "Service file created."
#+end_src

* Quick decision tree

- Service not starting → check =journalctl -u f2_crowd_scanner=
- License error → verify MAC address and re-activate
- Camera not detected → verify RTSP URL in config.ini
- FaceMe SDK error → check =$FACEME_HOME= environment variable
- Python import error → verify virtual environment activation
