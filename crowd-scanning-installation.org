* Context

This document provides a step-by-step installation guide for the *FACES 2 Crowd Scanning* application on NVIDIA Jetson Orin NX devices.

The recipe uses absolute paths and can be executed from any directory on the Jetson device.
All code blocks are designed to be idempotent where possible.

* Phase 5: License Activation

** Step 5.1: Get device MAC address

Provide this device identifier to your administrator to generate a license key.

#+begin_src bash :results output replace :exports both
  set -euo pipefail

  # Interface priority order for Jetson devices
  IF_ORDER=("eth1" "eth2" "eth0")

  is_valid_mac() {
    local mac="$1"
    local iface="${2:-}"

    # Must match xx:xx:xx:xx:xx:xx format
    [[ "$mac" =~ ^([0-9A-Fa-f]{2}:){5}[0-9A-Fa-f]{2}$ ]] || return 1

    # Normalize to lowercase
    mac="${mac,,}"

    # Reject invalid/dummy MACs
    [[ "$mac" != "00:00:00:00:00:00" ]] || return 1
    [[ "$mac" != "00:00:00:00:00:01" ]] || return 1
    [[ "$mac" != "ff:ff:ff:ff:ff:ff" ]] || return 1

    # Skip virtual interfaces (no physical device link)
    if [[ -n "$iface" ]] && [[ ! -e "/sys/class/net/$iface/device" ]]; then
      return 1
    fi

    return 0
  }

  echo "=== Network Interfaces ==="
  (ip link show | grep -E "^[0-9]+:|ether") 2>/dev/null || true

  echo ""
  echo "=== Device Identifier for License Request ==="

  for IF in "${IF_ORDER[@]}"; do
    addr_file="/sys/class/net/$IF/address"
    [[ -r "$addr_file" ]] || continue
    mac=$(<"$addr_file")
    if is_valid_mac "$mac" "$IF"; then
      mac_no_colon=${mac//:/}
      printf 'f2_%s\n' "${mac_no_colon,,}"
      exit 0
    fi
  done

  echo "Error: no valid MAC found on ${IF_ORDER[*]}." >&2
  exit 1
#+end_src

** Step 8.2: Create systemd service file

#+begin_src bash :results output replace :exports both :dir /sudo:: :session f2cs :var LICENSE_KEY=license-key
  set -euo pipefail

  sudo tee /etc/systemd/system/f2_crowd_scanner.service > /dev/null << SERVICEEOF
[Unit]
Description=F2 Crowd Scanning FaceMe Service
After=network.target
StartLimitBurst=3
StartLimitIntervalSec=60
OnFailure=f2_crowd_scanner-email@%n

[Service]
Type=simple
WorkingDirectory=$PROJECT_DIR
ExecStart=/usr/bin/script -q -c "$VENV_DIR/bin/python $PROJECT_DIR/main.py" /dev/null
Restart=always
RestartSec=10
Environment=FACEME_LICENSE_KEY=$LICENSE_KEY
StandardOutput=journal
StandardError=journal

[Install]
WantedBy=multi-user.target
SERVICEEOF

  echo "Service file created."
#+end_src

* Quick decision tree

- Service not starting → check =journalctl -u f2_crowd_scanner=
- License error → verify MAC address and re-activate
- Camera not detected → verify RTSP URL in config.ini
- FaceMe SDK error → check =$FACEME_HOME= environment variable
- Python import error → verify virtual environment activation
